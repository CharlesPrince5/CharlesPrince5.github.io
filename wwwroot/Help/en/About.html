V-Suite is a suite of Windows, web and mobile applications. V-Suite applications are made possible by open source software.


Compatibility


    • Data Transfer supports transfer from V-Suite CORE V9 and earlier.
    • V-Suite Web R4 provides services to Mobile Knowledge Book R3.


What's New in R4


General


    • Introduced Single Sign-On (SSO) capability, allowing users to sign in using an external SSO provider.
    • Data Transfer is now compatible with V-Suite CORE V9.


Knowledge Book Viewer


    • Implemented the Edge Shader, enhancing edge visibility for complex 3D models displayed in the viewer.
    • Enhanced the URL query strings with new actions.


What's New in R3 Update 4


System Administration


    • Users can view and export Audit Logs.


What's New in R3 Update 3


General


    • Users can configure their Locale Preferences.
    • Improved the Data Transfer performance for large projects.


Knowledge Book


    • Knowledge Book search includes an option to search for Scratch Pad Pages.
    • The Grid view search results view mode has been added to the Knowledge Book search.
    • The Knowledge Book search page has the ability to Search for all Knowledge Views, Folders or Scratch Pad Pages.
    • Scratch Pad Pages can be exported to PDF - either from the Scratch Pad Page Editor or directly from the Scratch Pad search results grid.


Knowledge Book Viewer


    • Added the ability to Hide Assets in the viewer.
    • Users can use a new Follow Camera mode to move the camera along a path defined by a Route.
    • The X-ray view mode helps to quickly identify what is picked in the viewer.
    • Added the Asset Context Menu and Navigation popup controls, to simplify performing actions on a picked asset.
    • Improved the layout and readability of the Asset Information panel, including a new option to hide inherited attributes from the data displayed on the panel.
    • Added the URL Links list, providing direct access to URL info contained in asset attributes.
    • Slide-out panels in the Knowledge Book Viewer can be set to auto-expand in the UI Preferences.
    • The graphical content of a Knowledge View can be exported into the GLB or OBJ formats.
    • Introduced the Ambient lighting mode.


What's New in R3 Update 2

V-Suite Web R3 is a minor release which introduces some performance improvements for CAD imported Graphical Data and compatibility with V-Suite CORE 2021 and Knowledge Book R3.


Knowledge Book


    • Improved the viewer performance for CAD imported Graphical Data (in particular the performance to render mesh-type graphical data).


Customers should note that this is the first of a series of planned Graphics performance improvements.


    • Added support to access Knowledge View Folders, Knowledge Views and Scratch Pad Pages via a URL query string.
    • Added the Knowledge View Information slide-out.
    • Added the ability to Zoom on a Scratch Pad Page.
    • Newly created Scratch Pad Pages are now Public by default.
