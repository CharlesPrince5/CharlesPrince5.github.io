Edit Transfer Job

In order to transfer data from V-Suite CORE to V-Suite Web, you must configure data transfer jobs (one for each V-Suite CORE project).  

You can edit an existing data transfer job via the Data Transfer application. The application can be accessed via the Project Administration icon on the V-Suite home page.



In the data transfer job table, select the data transfer job you wish to edit and click the Edit icon in the Action column. This displays the Edit Data Transfer Job form which lets you modify the following:

  • V-Suite CORE Connection
  • Job Schedule


V-Suite CORE Connection

Click the View Connection Properties button to display the V-Suite CORE connection information associated with the data transfer job.


Click the Edit Connection button to associate the data transfer job with a different V-Suite CORE connection.


You can only edit the V-Suite CORE connection when the data transfer job has been successfully run at least once. If the data for the job has not been transferred yet, the Edit Connection button will be disabled.


Edit Connection

The Edit Connection form lets you modify the V-Suite CORE connection associated with the data transfer job.


  • V-Suite CORE Connection - Select an existing V-Suite CORE connection or click the Add button to create a new connection alias.


Click Ok to confirm the new selection.


Selecting a new V-Suite CORE Connection will automatically locate the matching CORE project using the project ID.

  • If a matching CORE project is found, the V-Suite CORE Project Name will be updated with the name of the project stored on the V-Suite CORE server.
  • If no matching CORE project is found on the server, the V-Suite CORE Project Name will be set to blank and the ability to save the data transfer job will be disabled.


Note that it is not possible to manually update the V-Suite CORE Project Name or the V-Suite Project Name fields.
