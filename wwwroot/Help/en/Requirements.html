Requirements

To ensure optimal GPU performance when using V-Suite 3D web applications, browse the recommendations below and compare to your configuration.

Browsers

V-Suite currently supports these tested and approved browsers:

  • Mozilla Firefox
  • Google Chrome
  • Microsoft Edge

Mac OS Safari is currently not supported.

V-Suite WEB client no longer supports Microsoft Internet Explorer 11 due to Microsoft ending support for Internet Explorer (IE) 11.


WebGL

V-Suite 3D web and mobile applications require WebGL. Make sure your browser has WebGL enabled. On Chrome, it should be enabled by default, unless your graphics card does not support WebGL.

Enable WebGL for Chrome

  • Go to chrome://flags
  • Ensure that Disable WebGL is not activated (you'll need to relaunch Chrome for any changes to take effect)

Enable WebGL for Firefox

  • Go to about:config
  • Search for webgl.disabled
  • Ensure that its value is false (any changes take effect immediately without relaunching Firefox)


Enable WebGL for Microsoft Edge

  • Go to edge://settings/system
  • Ensure that "Use hardware acceleration when available" is checked. Restart your browser.
  • Go to edge://gpu
  • Under Graphics Feature Status find WebGL, ensure that Hardware Accelerated is listed.

Notes

If your graphics drivers are too old to support WebGL, you may be able to upgrade them.

More information is available here:

  • Chrome: Graphics aren't showing properly
  • Firefox: Upgrade your graphics drivers

Blacklists

Certain graphics cards and drivers are blacklisted by some browsers because they are known to contain serious bugs or cause crashes. In very severe cases, these can even cause the entire computer to freeze! 

Many browsers have the option to override the blacklist (ie: attempt to use the blacklisted features, despite the risks). This is not recommended! You should only override the blacklist if you understand and accept the risks.

More information is available here: WebGL Blacklists (https://www.khronos.org/webgl/wiki/BlacklistsAndWhitelists)
