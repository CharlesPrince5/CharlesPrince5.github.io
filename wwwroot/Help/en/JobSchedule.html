Job Schedule

About Job Schedule

Each Data Transfer Job can be associated with a recurring Job Schedule allowing the project administrator to automate the transfer of data from V-Suite CORE.

Job Schedule Information

The job schedule information consists of the following:

  • Enabled - Specifies whether the schedule is enabled for the transfer job. If the schedule is not enabled the data transfer will not automatically run. 
  • Days to Run - Specifies on which days the data transfer job must automatically run. To automatically run the data transfer on weekdays, select Mon, Tue, Wed, Thu and Fri.
  • Start Date - Specifies on which date the schedule should start transferring data.
  • Frequency - Specifies how often the data should be transferred on the days the transfer is scheduled to run. 
  • At (or Between) - Specifies the time at which the data transfer is run if you transfer data once or the start and end time between which the transfer is to be run.

Examples

I have a V-Suite CORE training database which I want to transfer but there is no need to automatically transfer its data.

Enabled - Not Checked. I can still manually run the transfer job but it will not automatically run.


I have a V-Suite CORE database which is not updated very often so just want its data to be transferred in the weekend when nobody works.


Enabled - Checked (Enable automatic transfer)
Days to Run - Sun (Transfer on Sunday)
Start Date - Today (Transfer next Sunday and each Sunday thereafter)
Frequency - Once (Transfer once each Sunday) 
At - 18:00 (Transfer at 6 o'clock in the afternoon)


I have a V-Suite CORE database which is updated frequently during weekdays but it is not crucial that those changes are immediately available.

Enabled - Checked (Enable automatic transfer)
Days to Run - Mon, Tue, Wed, Thu and Fri (Transfer every weekday)
Start Date - Today (Start transferring next weekday and every weekday thereafter)
Frequency - Once (Transfer once each weekday) 
At - 22:00 (Transfer at 10 o'clock at night)


I have a V-Suite CORE database which is updated frequently during weekdays but it is important that those changes are available as quickly as possible.


Enabled - Checked (Enable automatic transfer)
Days to Run - Mon, Tue, Wed, Thu and Fri (Transfer every weekday)
Start Date - Today (Start transferring next weekday and every weekday thereafter)
Frequency - Every 2 hours (Transfer every 2 hours on weekdays) 
Between - 09:00 and 19:00 (Transfer every 2 hours on weekdays between 9 o'clock in the morning and 7 o'clock in the afternoon)
