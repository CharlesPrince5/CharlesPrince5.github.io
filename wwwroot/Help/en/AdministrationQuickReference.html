Administration Quick Reference

This help topic will serve as a quick introduction into V-Suite administration functions. It will help you discover the main administration features and will help you find useful documentation links. This help documentation is organized in topics intended for individuals responsible for performing administrative tasks.

In V-Suite we distinguish between system administration tasks and project administration tasks. 

       

A system administrator typically performs IT related tasks and should be familiar with IIS installation and configuration, Active Directory and other IT related tasks. A Project Administrator closely works with the Organization and understands how the users in the organization use the software, and which users are granted access to the organization.
