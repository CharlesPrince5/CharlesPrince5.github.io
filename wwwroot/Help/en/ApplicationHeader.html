Application Header

All V-Suite web applications have a common header which provides typical application functionality.


The header provides the following functions:

What can I do?

What to expect 

Click on V-Suite logo

Jumps to the V-Suite home page

Click on Application Name

Jumps to the application main page

Click on Help icon 

Opens the application help menu.

Click on Share icon 

Copies or emails an inbound URL.

Exports the page content.

Prints the page content.


An inbound URL provides the user a quick way to access an application, Knowledge View, Knowledge View Query or an asset in a Knowledge View.

User Profile

The User Profile menu provides access to settings and actions related to the profile of the currently signed-in user.

What can I do?

What to expect

Click on My account

Opens the My Account form.

Click on Sign out

Allows the current user to sign-out from V-Suite.

Click on Locale \ My Preferences

Opens the Locale Preferences form.


My Account form

The My Account form presents you with detailed information about your V-Suite user account.


Clicking the Change Password button brings up a form in which you can change the password of your V-Suite user account. To change your password, you need to enter the following information:


  • Old Password - Your old password for the V-Suite account
  • New Password - The new password for the V-Suite account. The password must meet the password complexity settings.
  • Confirm New Password - For confirmation, enter the desired new password again.


Once done entering the password info, clicking the Change button will change your password. You can cancel the request to change password by clicking Cancel.

Locale Preferences form

The Locale Preferences form allows you to set up your language, date, and time preferences for the V-Suite web applications:


  • Language - Choose the preferred language for the V-Suite Web applications.
    • Supported languages are: English, Chinese (simplified), Japanese
    • If left at Default, your web browser's language settings are used to determine the language, with English being selected in case the preferred web browser's language is not in the list of languages supported by V-Suite web.
  • Date pattern - Select how to display dates.
    • The choice of available date patterns is limited based on the selected language.
  • Time pattern - Select how to display times.
    • The choice of available time patterns is limited based on the selected language.
  • Time zone - Configure the time zone used when displaying date and time values.
    • If left at Default, the local time zone of your machine will be used.


Once done configuring the Locale Preferences, click the Save button to confirm your new selections. You can also cancel the changes by clicking Cancel.
