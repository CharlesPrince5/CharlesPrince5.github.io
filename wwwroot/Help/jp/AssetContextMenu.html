アセットコンテキストメニュー

アセットが選択されている(ビューアでハイライト表示されている) 場合、ビューアエリア内の任意の場所を右クリックしてアセットのコンテキスト メニューを表示できます。


アセットのコンテキスト メニューには次のコマンドが用意されています。

選択したアセットの境界 (範囲) にズームします。

選択したアセットをビューアで非表示にします。非表示のアセットは、 [非表示のアセット] パネルに再表示できます。

マークされたポイントが選択されている場合、このコマンドは使用できません。

選択したアセットのアセット情報を表示します。

コンテンツ ブラウザで現在のアセットを選択します。

マークされたポイントが選択されている場合、このコマンドは使用できません。

選択したアセットの属性のURL リンクを表示します。

選択したアセットまたはその子孫にアタッチされたマークされたポイントを表示します。このコマンドは、アタッチされマークされたポイントをリストし、リストからマークされたポイントの 1 つを選択できます。

マークされたポイントが選択されている場合、このコマンドは使用できません。

選択したアセットのURL をクリップボードにコピーします。アセットの受信 URL は、ナレッジビューメモ、電子メール、またはインターネット ブラウザのアドレス バーに貼り付けることができます。
アセット受信 URL は、アセットURL のコピー元のナレッジビューを開き、アセットの境界 (範囲) にズームして、ビューアでアセットを選択します。

アセットメモを追加します。このコマンドは、新しいナレッジビューノートを作成し、アセットURL をノートにコピーします。このコマンドは、アセットの名前をメモの名前とメモ内のアセットURL のテキストに割り当てます。

選択したアセットを解除し、現在の選択を事実上クリアし、ナビゲーション ポップアップを閉じます。


URLリンク

[URL リンク] サブメニューには、現在選択されているアセットの属性(URL 属性タイプであり、有効な URL 値が割り当てられているもの) がすべて表示されます。


URL 属性値を含む属性セットが 1 つだけある場合、URL リンクは 1 つのリストに表示されます。


URL 属性値を持つ複数の属性セットが存在する場合、表示される URL リンクは、属性セットにちなんで名付けられたサブフォルダーにグループ化されます。


URL リンクをクリックすると、その URL がインターネット ブラウザの新しいタブで開きます。

URL リンク リストには、有効なhttp アドレスとhttpsアドレスのみが表示されます。他のプレフィックス ( file 、 ftpなど) は含まれません。
