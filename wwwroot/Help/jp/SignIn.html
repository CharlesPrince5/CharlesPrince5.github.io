ログイン

V-Suite Web アプリケーションを利用するには、システムにサインインする必要があります。アクセスは、有効な登録ユーザーにのみ許可されます。認証されていないユーザーは、 V-Suiteサインイン ページにリダイレクトされます。新しいユーザーを追加する方法については、 「ユーザーの追加」トピックを参照してください。


システムはユーザーを検証する 2 つの方法をサポートしています。

  • シングル サインオン (SSO) 認証
  • V-Suite (パスワードベース) 認証


システム管理者によって有効にされた認証方法のみを使用できます。


シングル サインオン (SSO) 認証

次の手順では、Azure AD が SSO プロバイダーとして機能する場合の SSO 認証プロセスについて説明します。 Active Directory フェデレーション サービス (ADFS) のプロセスは非常によく似たフローに従うことに注意してください。

  • 外部 SSO プロバイダー (Azure AD) を使用してサインインするには、 [企業アカウントでサインイン]ボタンをクリックします。
  • Azure AD ログイン ページにリダイレクトされます。すでに Azure AD にサインインしている場合 (たとえば、別のアプリケーションから)、このページが表示されない可能性があります。
  • Azure AD ログイン ページが表示された場合は、Azure AD ユーザー名 (通常は職場の電子メール)、Azure AD パスワードを入力し、[サインイン] をクリックします。組織が多要素認証 (MFA) を有効にしている場合は、プロンプトに従って追加の検証を行ってください。
  • Azure AD で正常に認証されると、 V-Suite Web アプリケーションにリダイレクトされます。 Azure AD アカウントがV-Suiteユーザー アカウントにリンクされている場合は、 V-Suiteアプリケーションにアクセスできるようになります。
  • Azure AD アカウントがV-Suiteユーザー アカウントにリンクされていない場合は、 V-Suite管理者に連絡するか、別のユーザーとしてサインインすることを求めるエラー メッセージが表示されます。


V-Suite認証

V-Suite認証を使用してサインインするには、 [V-Suiteアカウントでサインイン]ボタンを選択します。次に、 V-Suite認証情報を入力するよう求められます。

  • 電子メール - V-Suiteアカウントに関連付けられた電子メール アドレスを入力します。例: 「johndoe@mydomain.com」。
  • パスワード - V-Suiteアカウントのパスワードを入力します。


サインイン - パスワードの変更

初めてV-Suiteにサインインするとき、または期限切れのパスワードを使用してV-Suiteにサインインしようとするときは、新しいV-Suiteアカウントのパスワードを入力するように求められます。パスワードを変更するには、次の情報を入力する必要があります。

  • 古いパスワード - V-Suiteアカウントの古い (期限切れの) パスワード
  • 新しいパスワード - V-Suiteアカウントの新しいパスワード。パスワードはパスワードの複雑さの設定を満たしている必要があります。
  • 新しいパスワードの確認 - 確認のため、希望の新しいパスワードを再度入力します。


パスワード情報の入力が完了したら、 「パスワードの変更」ボタンをクリックするとパスワードが変更され、 V-Suite Web にサインインします。


ログイン状態を保持する

[サインインしたままにする] チェックボックスをオンにすると、ブラウザを閉じた後でも、情報は最大 30 日間記憶されます。次回 30 日以内に同じブラウザーを開くと、自動的にサインインされます。「Remember Me」を機能させるには、アプリケーションの「サインアウト」ボタンをクリックせずに、終了後にブラウザーを閉じる必要があります。

[サインインしたままにする] オプションを使用すると、再度サインインすることが簡単になりますが、これは、コンピューターにアクセスできるすべてのユーザー (同じアカウントを使用してコンピューターにアクセスする) がV-Suite を使用してサインインできることも意味します。アカウント。公共のコンピューターやアクセスを制御できない場所では、「Remember Me」を使用しないでください。


V-Suite管理者は、サインイン状態を保持する機能を完全に無効にしたり、サインイン情報を記憶する日数 (デフォルトでは 30 日) を調整したりすることができます。
