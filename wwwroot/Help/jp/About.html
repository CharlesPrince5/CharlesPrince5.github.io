    V-Suite Web R4
    ビルド バージョン: 2.2.4.210

    Friday, July 7, 2023


    Visionaize、 V-Suite® および 3D アセット Integrity Management™ は、Visionaize, Inc. の登録商標および商標です。記載されているその他すべての製品およびサービス名は、それぞれの会社の商標です。ここに記載されている実際の会社名および製品名は、それぞれの所有者の商標である場合があります。


    V-Suite は、Windows、Web、およびモバイル アプリケーションのスイートです。 V-Suite アプリケーションは、オープン ソース ソフトウェアV-Suite実現されています。


    互換性


      • データ転送は、 V-Suite CORE V9 以前からの転送をサポートしています。
      • V-Suite Web R4 は、 Mobile Knowledge Book R3 にサービスを提供します。


    R4 の新機能


    全般的


      • シングル サインオン (SSO) 機能が導入され、ユーザーが外部 SSO プロバイダーを使用してサインインできるようになりました。
      • データ転送がV-Suite CORE V9 と互換性を持つようになりました。


    Knowledge Bookビューア


      • エッジ シェーダを実装し、ビューアに表示される複雑な 3D モデルのエッジの可視性を強化しました。
      • 新しいアクションでURL クエリ文字列が強化されました。


    R3 Update 4 の新機能


    システム管理


      • ユーザーは監査ログを表示およびエクスポートできます。


    R3 Update 3 の新機能


    全般的


      • ユーザーはロケール設定を構成できます。
      • 大規模プロジェクトのデータ転送パフォーマンスが向上しました。


    Knowledge Book


      • Knowledge Bookの検索には、スクラッチ パッド ページを検索するオプションが含まれています。
      • グリッドビューの検索結果表示モードがKnowledge Book検索に追加されました。
      • Knowledge Bookの検索ページには、すべてのナレッジビュー、フォルダ、またはスクラッチ パッド ページを検索する機能があります。
      • スクラッチ パッド ページは、スクラッチ パッド ページ エディターから、またはスクラッチ パッド検索結果グリッドから直接PDF にエクスポートできます。


    Knowledge Bookビューア


      • ビューアでアセットを非表示にする機能が追加されました。
      • ユーザーは、新しいカメラ追従モードを使用して、 Routeで定義されたパスに沿ってカメラを移動できます。
      • X線ビュー モードは、ビューアで何が選択されているかをすばやく識別するのに役立ちます。
      • 選択したアセットに対するアクションの実行を簡素化するために、アセットコンテキスト メニューとナビゲーション ポップアップコントロールを追加しました。
      • 「アセット情報」パネルのレイアウトと読みやすさが改善され、パネルに表示されるデータから継承された属性を非表示にする新しいオプションが追加されました。
      • URL リンクリストが追加され、アセット属性に含まれる URL 情報に直接アクセスできるようになりました。
      • Knowledge Bookビューアのスライドアウト パネルは、 UI 設定で自動展開するように設定できます。
      • ナレッジビューのグラフィック コンテンツは、 GLB または OBJ 形式にエクスポートできます。
      • アンビエント照明モードが導入されました。


    R3 Update 2 の新機能

    V-Suite Web R3 は、CAD でインポートされたグラフィック データのパフォーマンス向上と、 V-Suite CORE 2021 およびKnowledge Book R3 との互換性を導入したマイナー リリースです。


    Knowledge Book


      • CAD からインポートされたグラフィック データのビューアのパフォーマンス (特に、メッシュ タイプのグラフィック データをレンダリングするパフォーマンス) が向上しました。

    お客様は、これが計画されている一連のグラフィックス パフォーマンスの改善の最初のものであることに注意してください。

      • URL クエリ文字列を介してナレッジビューフォルダー、ナレッジビュー、およびスクラッチ パッド ページにアクセスするためのサポートが追加されました。
      • 「ナレッジビュー情報」スライドアウトを追加しました。
      • スクラッチパッドページをズームする機能が追加されました。
      • 新しく作成されたスクラッチ パッド ページは、デフォルトで公開されるようになりました。
