资产上下文菜单

选取资产（在查看器中突出显示）后，您可以右键单击查看器区域内的任意位置以显示资产上下文菜单。


资产上下文菜单提供以下命令：

缩放至所选资源的边界（范围）。

在查看器中隐藏选定的资源。隐藏的资源可以在隐藏资源面板中重新显示。

如果选取了标记点，则此命令不可用。

显示所选资产的资产信息。

在内容浏览器中选择当前资产。

如果选取了标记点，则此命令不可用。

查看所选资产属性的URL 链接。

查看所选资产或其后代的附加标记点。该命令列出了附加的标记点，并允许您从列表中选择标记点之一。

如果选取了标记点，则此命令不可用。

将选取的资源 URL 复制到剪贴板。资产入站 URL 可以粘贴到知识视图注释、电子邮件或互联网浏览器地址栏中。
资产入站 URL 将打开从中复制资产 URL 的知识视图，缩放到资产的边界（范围）并在查看器中选择资产。

添加资产注释。此命令创建一个新的知识视图注释并将资产 URL 复制到该注释中。该命令将资产名称分配给注释名称以及注释内资产 URL 的文本。

取消选取所选取的资源，有效地清除当前选择并关闭导航弹出窗口。


网址链接

URL 链接子菜单显示当前选定资产的所有属性，这些属性属于 URL 属性类型并且分配了有效的 URL 值。


如果只有一个包含 URL 属性值的属性集，则 URL 链接将显示在单个列表中：


如果存在多个具有 URL 属性值的属性集，则显示的 URL 链接将分组到以属性集命名的子文件夹中：


单击 URL 链接将在新的互联网浏览器选项卡中打开该 URL。

URL 链接列表仅显示有效的http和https地址。不包括其他前缀（ file 、 ftp等）。
