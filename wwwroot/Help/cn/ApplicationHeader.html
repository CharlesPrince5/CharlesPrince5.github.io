我的帐户表格

“我的帐户”表单向您提供有关您的V-Suite用户帐户的详细信息。


单击“更改密码”按钮会弹出一个表单，您可以在其中更改V-Suite用户帐户的密码。要更改密码，您需要输入以下信息：

  • 旧密码 - 您的V-Suite帐户的旧密码
  • 新密码 - V-Suite帐户的新密码。密码必须符合密码复杂度设置。
  • 确认新密码 - 再次输入所需的新密码进行确认。


输入密码信息后，单击更改按钮将更改您的密码。您可以通过单击“取消”来取消更改密码的请求。

区域设置首选项表格

“区域设置首选项”表单允许您设置V-Suite Web 应用程序的语言、日期和时间首选项：

  • 语言 - 选择V-Suite Web应用程序的首选语言。
    • 支持的语言有：英语、中文（简体）、日语
    • 如果保留为“默认” ，则将使用您的 Web 浏览器的语言设置来确定语言，如果首选 Web 浏览器的语言不在V-Suite Web 支持的语言列表中，则会选择英语。
  • 日期模式 - 选择如何显示日期。
    • 可用日期模式的选择根据所选语言而受到限制。
  • 时间模式 - 选择如何显示时间。
    • 可用时间模式的选择根据所选语言而受到限制。
  • 时区 - 配置显示日期和时间值时使用的时区。
    • 如果保留为Default ，则将使用您计算机的本地时区。


完成区域设置首选项的配置后，单击“保存”按钮确认您的新选择。您还可以通过单击“取消”来取消更改。
