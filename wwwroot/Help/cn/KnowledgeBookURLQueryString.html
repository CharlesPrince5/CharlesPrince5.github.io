Knowledge BookURL 查询字符串

用户可以通过应用程序标题中的共享图标共享知识视图或文件夹。用户还可以生成入站 URL，指示Knowledge Book应用程序打开知识视图并通过资产上下文菜单缩放到该知识视图视图中的特定资产 复制链接命令。这些应用程序命令将根据内部 ID 生成入站 URL。应用程序生成的入站 URL 将如下所示：


https://vsuite.mycompany.com/KnowledgeBook/rc/nav/122ace4d-31c0-e811-8166-0299893c3555


但是，为了允许外部网站或应用程序根据名称将用户引导至V-Suite知识视图或文件夹，可以提供带有查询字符串的 URL。

URL 查询字符串

Knowledge Book应用程序支持查询字符串来打开知识视图文件夹、知识视图或便笺簿页面。支持以下查询字符串：

操作区分大小写。指定“openkv”或“zoom”将起作用，“OpenKV”或“Zoom”将导致错误。

行动

描述

价值

?openfolder

导航到知识视图文件夹。

文件夹的编码位置。为了避免与文件夹和视图名称中的常见字符发生冲突，必须使用以下转义序列来指示文件夹分隔符：_._

?openkv

在查看器中打开知识视图。

知识视图的编码位置和名称

&spp

在便签本页面编辑器中打开便签本页面。


必须与?openkv动作配合使用。

暂存器页面的编码名称。

&zoom

缩放并选取（突出显示）查看器中的资源。


必须与?openkv动作配合使用。

资产的编码名称。

&type

类型是用于&zoom操作的可选补充参数。如果打开的知识视图中有多个具有特定名称的资产，它有助于唯一地标识资产。

资产的编码类型。

&pickcolor

Pickcolor 是用于&zoom操作的可选补充参数。它用于在缩放到查看器中的资源时指定替代资源突出显示颜色。

突出显示颜色的十六进制代码（没有初始标签“#”字符）。例如，指定 &pickcolor 值“8B0000”表示使用深红色作为突出显示颜色。

&viewer

控制所显示的知识视图应用程序的布局。


仅当您在 HTML IFrame 内渲染查看器时想要限制 Knowledge View 应用程序中显示的 UI 时，此操作才有用。


此操作仅供应用程序集成商使用。


必须与?openkv动作配合使用。

设置为=full可打开标准知识视图布局，其中包含所有控件和功能。当省略查看器操作时，这是默认选项。


设置为=basic可将 UI 和功能限制为仅适合嵌入 IFrame 内部时显示的 UI 和功能。限制包括：

  • 标准应用程序标题栏被隐藏。
  • 左侧选项卡控件（内容浏览器、显示管理器、注释和便签本）不可用。
  • 用于创建笔记和便签本页面的命令不可用。


设置为=navonly以打开具有最少（仅导航）功能的查看器。

&xray

打开查看器中的 X 射线模式。


必须与?openkv动作配合使用。

  • 设置为=on以打开 X 射线模式。
  • 设置为=off以关闭 X 射线模式。 （默认情况下，所有 KV 的 X 射线模式均处于关闭状态。）

文件夹、视图和资产名称不能包含 URI 保留字符或 Unicode 字符，必须进行编码（根据 RFC 3986）。例如，名称中的空格必须编码为“%20”。

  • 如果指定的文件夹、知识视图或便签本页面名称不存在或其名称不唯一，您将看到以下错误消息：


  • 如果指定的资产名称在知识视图中不存在或其名称在知识视图中不唯一，则应用程序将打开知识视图，但不会缩放到该资产。相反，内容浏览器将展开，并且不明确的资产名称将添加到资产搜索文本框。通过单击资产搜索文本框，您将看到匹配资产的列表：



打开文件夹示例：


在主机“ vsuite.mycompany.com ”上打开文件夹位置“ Amine Demo > Inspection ” ：

网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openfolder=Amine%20Demo_._Inspection

 

打开知识视图示例：


在主机“ vsuite.mycompany.com ”上，在文件夹位置“ Amine Demo > Inspection ”中打开知识视图“ Inspection Due Status ” ：

网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openkv=Amine%20Demo_._Inspection_._Inspection%20Due%20Status


打开便签本页面示例：


在主机“ vsuite.mycompany.com ”上，在文件夹位置“ Amine Demo > Inspection ”中的知识视图“Inspection Due Status”中打开便笺簿页面“ Field Notes ” ：

网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openkv=Amine%20Demo_._Inspection_._Inspection%20Due%20Status&spp=FIELD%20NOTES


缩放至资产示例：


在主机“ vsuite.mycompany.com ”上，在文件夹位置“ Amine Demo > Inspection ”的知识视图“ Inspection Due Status ”中缩放到名称为“ TK-0001 ”的资产：

网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openkv=Amine%20Demo_._Inspection_._Inspection%20Due%20Status&zoom=TK-0001


缩放至资产和类型示例：


在主机“ vsuite.mycompany.com ”上，在文件夹位置“ Amine Demo > Inspection ”的知识视图“ Inspection Due Status ”中缩放到名称为“ T-0001 ”的资产。由于有两个名为“ T-0001 ”的资源，一个设备组合，一个是设备主体，因此我们使用可选的“类型”参数来进一步确定我们是否有兴趣缩放到设备组合。

网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openkv=Amine%20Demo_._Inspection_._Inspection%20Due%20Status&zoom=T-0001&type=设备组合


使用 Pickcolor 和 XRay 模式缩放至资源示例：


在主机“ vsuite.mycompany.com ”上，在文件夹位置“ Amine Demo > Inspection ”的知识视图“ Inspection Due Status ”中缩放到名称为“ TK-0001 ”的资产。该资源将使用自定义（深红色）颜色突出显示。所有其他资产将使用灰色透明外观显示，类似于 X 射线图像。


网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openkv=Amine%20Demo_._Inspection_._Inspection%20Due%20Status&zoom=TK-0001&pickcolor=8B0000&xray=on


在基本查看器示例中打开知识视图：


在主机“ vsuite.mycompany.com ”上，使用有限的知识视图布局，在文件夹位置“ Amine Demo > Inspection ”中打开知识视图“ Inspection Due Status” ：

网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openkv=Amine%20Demo_._Inspection_._Inspection%20Due%20Status&viewer=basic


在最小（仅限导航）查看器中打开知识视图示例：


在主机“ vsuite.mycompany.com ”上，使用最小的知识视图布局，在文件夹位置“ Amine Demo > Inspection ”中打开知识视图“Inspection Due Status ” ：

网址：   https://vsuite.mycompany.com/KnowledgeBook/rc/?openkv=Amine%20Demo_._Inspection_._Inspection%20Due%20Status&viewer=navonly
