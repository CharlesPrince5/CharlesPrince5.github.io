    Visionaize、 V-Suite® 和 3D Asset Integrity Management™ 是 Visionaize, Inc. 的注册商标和商标。提及的所有其他产品和服务名称均为其各自公司的商标。本文提及的实际公司和产品的名称可能是其各自所有者的商标。


    V-Suite是一套 Windows、Web 和移动应用程序。 V-Suite 应用程序是通过开源软件V-Suite的。


    兼容性


      • 数据传输支持从V-Suite CORE V9 及更早版本的传输。
      • V-Suite Web R4为移动Knowledge BookR3提供服务​​。


    R4 的新增功能


    一般的


      • 引入了单点登录 (SSO) 功能，允许用户使用外部 SSO 提供商登录。
      • 数据传输现在与V-Suite CORE V9 兼容。


    Knowledge Book浏览器


      • 实施了边缘着色器，增强了查看器中显示的复杂 3D 模型的边缘可见性。
      • 使用新操作增强了URL 查询字符串。


    R3 更新 4 中的新增功能


    系统管理


      • 用户可以查看和导出审核日志。


    R3 更新 3 中的新增功能


    一般的


      • 用户可以配置他们的区域设置首选项。
      • 提高了大型项目的数据传输性能。


    Knowledge Book


      • Knowledge Book搜索包括搜索便签本页面的选项。
      • Knowledge Book搜索中添加了网格视图搜索结果查看模式。
      • Knowledge Book搜索页面能够搜索所有知识视图、文件夹或便笺簿页面。
      • 便笺本页面可以从便笺本页面编辑器或直接从便笺本搜索结果网格导出为 PDF 。


    Knowledge Book浏览器


      • 添加了在查看器中隐藏资产的功能。
      • 用户可以使用新的跟随相机模式沿着Route定义的路径移动相机。
      • X射线查看模式有助于快速识别查看器中所拾取的内容。
      • 添加了资产上下文菜单和导航弹出控件，以简化对所选资产的执行操作。
      • 改进了“资产信息”面板的布局和可读性，包括一个用于隐藏面板上显示的数据继承属性的新选项。
      • 添加了URL 链接列表，提供对资产属性中包含的 URL 信息的直接访问。
      • Knowledge Book查看器中的滑出面板可以在UI 首选项中设置为自动展开。
      • 知识视图的图形内容可以导出为 GLB 或 OBJ 格式。
      • 引入了环境照明模式。


    R3 更新 2 中的新增功能

    V-Suite Web R3 是一个次要版本，引入了 CAD 导入图形数据的一些性能改进以及与V-Suite CORE 2021 和Knowledge BookR3 的兼容性。


    Knowledge Book


      • 改进了 CAD 导入图形数据的查看器性能（特别是渲染网格类型图形数据的性能）。

    客户应注意，这是一系列计划的图形性能改进中的第一个。

      • 添加了通过URL 查询字符串访问知识视图文件夹、知识视图和便签本页面的支持。
      • 添加了知识视图信息滑出式。
      • 添加了缩放便签本页面的功能。
      • 新创建的便笺本页面现在默认为公共。
