要求

为了确保在使用V-Suite 3D Web 应用程序时获得最佳 GPU 性能，请浏览下面的建议并与您的配置进行比较。

浏览器

V-Suite目前支持以下经过测试和批准的浏览器：

  • 火狐浏览器
  • 谷歌浏览器
  • 微软边缘

目前不支持 Mac OS Safari。

由于 Microsoft 终止对 Internet Explorer (IE) 11 的支持， V-Suite WEB 客户端不再支持 Microsoft Internet Explorer 11。


网页GL

V-Suite 3D Web 和移动应用程序需要 WebGL。确保您的浏览器启用了 WebGL。在 Chrome 上，默认情况下应启用它，除非您的显卡不支持 WebGL。

为 Chrome 启用 WebGL

  • 转到 chrome://flags
  • 确保禁用 WebGL 未激活（您需要重新启动 Chrome 才能使更改生效）

为 Firefox 启用 WebGL

  • 转到关于：配置
  • 搜索 webgl。禁用
  • 确保其值为 false（任何更改都会立即生效，无需重新启动 Firefox）


为 Microsoft Edge 启用 WebGL

  • 转到边缘://设置/系统
  • 确保选中“可用时使用硬件加速”。重新启动浏览器。
  • 转到边缘://gpu
  • 在“图形功能状态”下找到“WebGL”，确保列出“硬件加速”。

笔记

如果您的图形驱动程序太旧而无法支持 WebGL，您也许可以升级它们。

更多信息请点击这里：

  • Chrome：图形无法正确显示
  • Firefox：升级您的图形驱动程序

黑名单

某些显卡和驱动程序被某些浏览器列入黑名单，因为已知它们包含严重错误或导致崩溃。非常严重的情况下，甚至会导致整个电脑死机！

许多浏览器都可以选择覆盖黑名单（即：尝试使用黑名单中的功能，尽管存在风险）。不推荐这样做！仅当您了解并接受风险时，您才应覆盖黑名单。

更多信息请参见： WebGL 黑名单 (https://www.khronos.org/webgl/wiki/BlacklistsAndWhitelists)
