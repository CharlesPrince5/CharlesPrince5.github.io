工作诊断信息

V-Suite CORE传输作业需要SQL Server Integration Services (SSIS) 。执行传输的 SSIS 包会将信息记录到诊断中。

要查找相关诊断记录，请单击诊断应用程序中的数据传输过滤器按钮。



每个作业都会生成具有相同会话 ID 的诊断记录。诊断将包含作业开始和完成的信息级别记录，可能还包含传输警告的警告级别记录和错误级别记录。
